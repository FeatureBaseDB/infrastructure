cfn-production
==============

This is a Cloudformation template generator for managing the production environment.

Usage
-----

### Create a stack

```
$ make create-stack STACK=foo
```

### Update a stack

```
$ make update-stack STACK=foo
```

### Delete a stack

```
$ make delete-stack STACK=foo
```

### Describe a stack

```
$ make describe-stack STACK=foo
```
